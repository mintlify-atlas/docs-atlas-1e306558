---
title: "Inicio rápido"
description: "Pon en marcha el Portal Sindical SIUT-ITSM en minutos: clona, instala dependencias, configura el entorno e inicia el servidor."
---

Esta guía te lleva desde cero hasta una instancia funcional del portal en el menor tiempo posible.

<Steps>
  <Step title="Clonar el repositorio">
    Clona el repositorio oficial y entra al directorio del proyecto:

    ```bash
    git clone https://github.com/mrtnsnchz/ost-siut-itsm.git
    cd ost-siut-itsm
    ```
  </Step>

  <Step title="Instalar dependencias de PHP">
    Asegúrate de tener **PHP 8.3+** y **Composer** instalados, luego ejecuta:

    ```bash
    composer install
    ```

    Esto instalará todas las dependencias declaradas en `composer.json` (Latte, PHP-DI, Monolog, PHPMailer, dompdf, etc.).
  </Step>

  <Step title="Configurar las variables de entorno">
    Copia el archivo de ejemplo y edítalo con tus credenciales:

    ```bash
    cp .env.example .env
    ```

    Abre `.env` y ajusta al menos los valores de base de datos y URL base:

    ```ini
    DB_HOST=localhost
    DB_NAME=siut_itsm
    DB_USER=tu_usuario
    DB_PASSWORD=tu_contraseña
    DB_PORT=3306

    BASE_URL=http://localhost
    APP_ENV=dev
    ```

    Consulta [Variables de entorno](/setup/environment) para la referencia completa de cada variable.
  </Step>

  <Step title="Crear e importar la base de datos">
    Crea la base de datos en MySQL / MariaDB e importa el esquema:

    ```bash
    mysql -u tu_usuario -p -e "CREATE DATABASE siut_itsm CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;"
    mysql -u tu_usuario -p siut_itsm < database.sql
    ```

    Opcionalmente, carga datos de ejemplo:

    ```bash
    mysql -u tu_usuario -p siut_itsm < database.seed.sql
    ```
  </Step>

  <Step title="Iniciar el servidor de desarrollo">
    Levanta el servidor embebido de PHP en el puerto `8888`:

    ```bash
    composer run servidor
    ```

    Abre tu navegador en [http://localhost:8888](http://localhost:8888).

    <Note>
      El script `servidor` ejecuta `php -S localhost:8888 -t .` definido en `composer.json`. Es adecuado únicamente para desarrollo local.
    </Note>
  </Step>
</Steps>
