# SIUT-ITSM Portal Sindical

El **Portal Sindical SIUT-ITSM** es la plataforma digital oficial del Sindicato Único de Trabajadores del Instituto Tecnológico Superior de Macuspana. Ofrece a los agremiados y personal directivo un espacio centralizado para gestionar trámites, consultar información sindical y comunicarse con la organización.

## ¿Qué es SIUT-ITSM?

SIUT-ITSM es una aplicación web desarrollada en **PHP 8.3+** que sirve como portal oficial para los integrantes del sindicato. Permite gestionar trámites internos, publicar noticias y comunicados, administrar préstamos, y mantener un directorio de agremiados con sus documentos e información laboral.

## Tecnologías

| Tecnología | Propósito |
|---|---|
| PHP 8.3+ | Lenguaje de backend principal |
| Latte 3.x | Motor de plantillas para las vistas |
| PHP-DI 7.x | Contenedor de inyección de dependencias |
| Monolog 3.x | Sistema de logging |
| PHPMailer 6.x | Envío de correos electrónicos |
| dompdf 3.x | Generación de documentos PDF |
| Pest 4.x | Framework de pruebas |
| MySQL / MariaDB | Base de datos relacional |
| Docker | Contenedores para despliegue |

## Estructura de navegación

El portal tiene dos áreas principales:

- **Área pública** (`/public/`) — Accesible sin autenticación: página principal, publicaciones, transparencia, contacto y preguntas frecuentes.
- **Portal privado** (`/public/portal/`) — Requiere autenticación. Muestra contenido personalizado según el rol del usuario.
